# TicTocToe
- เป็นเกมส์แบบที่ต้องใช้ผู้เล่น 2 คน

จุดประสงค์
- เพื่อให้ผู้ใช้งานเล่นเกมส์กับเพื่อน
- เพื่อให้ผู้พัฒนาผึกเขียนโปรแกรม

หน้าต่างโปรแกรม
  กรอกชื่อผู้เล่นทั้ง 2 คน
  
  ผู้เล่นคนที่ 1 เป็นตัวอักษร X
  
  ผู้เล่นคนที่ 2 เป็นตัวอักษร O
